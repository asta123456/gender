# Gender Classification using Machine Learning

This project is a Streamlit web application that performs Gender Classification using Machine Learning.

The application allows users to:

- Upload a CSV dataset
- Train a Logistic Regression model
- View model accuracy
- Display confusion matrix
- Display classification report
- Predict gender using custom input values

---

# Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
